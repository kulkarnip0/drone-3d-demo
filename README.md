# Drone 3D Mission World

A simple browser-based 3D demo for visualizing a drone mission world.

It includes:

- Flat terrain
- Roads
- Small buildings
- Trees
- Mission zones
- Drone paths
- Three moving drones

## How to run on Windows

1. Install VS Code.
2. Install the VS Code extension called **Live Server**.
3. Open this folder in VS Code.
4. Right-click `index.html`.
5. Click **Open with Live Server**.

The demo opens in your browser.

## Why this demo exists

This is an early investor/demo prototype to show a simple operational 3D world where multiple drones can fly predefined missions.

Later this can be extended with:

- Drone coordination
- Mission planning
- Zone coverage logic
- Threat/target detection mockups
- Global belief map visualization
- Real drone telemetry using MAVLink
